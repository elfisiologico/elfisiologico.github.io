# Fuentes PDF del repositorio científico

Archivo privado en iCloud para los textos completos empleados en la evaluación científica de FisioLógico.

## Estructura

- `pdfs-originales/`: versión definitiva del artículo evaluado.
- `material-suplementario/`: anexos, protocolos, tablas suplementarias y correcciones.
- El inventario de PDF todavía necesarios está en `../docs/solicitud-pdfs-repositorio.md`.

## Convención

Nombrar el original como `PMID.pdf`, por ejemplo `41489531.pdf`. Si no existe PMID, utilizar `DOI_normalizado.pdf`. Los suplementos deben comenzar con el mismo identificador, por ejemplo `41489531_suplemento-1.pdf`.

Estos archivos son fuentes de trabajo privadas. No se copian a `repositorio/`, no se enlazan públicamente y no deben subirse al despliegue web. La ficha pública enlaza siempre a PubMed, DOI o al editor original.
