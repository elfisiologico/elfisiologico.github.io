Decisión rápida

Útil para una visión transversal del miembro superior.

QuickDASH permite seguir síntomas y dificultad funcional cuando el problema afecta al brazo, hombro o mano y se busca una medida breve.

No localiza una lesión, no separa automáticamente el peso de cada articulación y no sustituye una valoración clínica.

Aplicación

Once ítems y una regla de puntuación específica.

La puntuación principal se transforma a una escala de 0 a 100. El cálculo requiere al menos 10 de los 11 ítems contestados. Existen dos módulos opcionales de cuatro ítems para trabajo y para deporte o actividad artística, que se puntúan por separado.

Interpretación

Una puntuación mayor indica mayor dificultad.

El cambio debe interpretarse con datos de error de medida y relevancia para la población concreta. No debe atribuirse el resultado a hombro, codo, muñeca o mano sin otras fuentes de información.

No intercambies DASH y QuickDASH

La versión corta no es simplemente una hoja abreviada para modificar libremente. Tiene instrucciones, reglas y propiedades propias.

Versión española

Existe una traducción oficial para España.

El Institute for Work & Health publica una versión española de España. Debe diferenciarse de otras traducciones regionales y utilizarse sin modificar palabras, opciones o algoritmo.

Uso y derechos

Gratuito para determinados usos; no libre de condiciones.

El IWH permite sin coste el uso clínico para tratar o valorar pacientes, la investigación no comercial y otros usos sin ánimo de lucro, siempre bajo sus condiciones. La incorporación a productos comerciales o software requiere autorización y puede requerir licencia.

FisioLógico no reproduce los ítems; enlaza la versión y las condiciones oficiales.
